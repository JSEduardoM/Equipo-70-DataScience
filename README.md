## üìñ Data Dictionary

| Column                  | Description                                                    |
| ----------------------- | -------------------------------------------------------------- |
| **order_id**            | Unique identifier for each order (e.g., ORD5001)               |
| **customer_id**         | Unique identifier for each customer (e.g., CUST1001)           |
| **age**                 | Customer age in years (e.g., 32)                               |
| **product_id**          | Unique identifier for each product (e.g., PROD210)             |
| **country**             | Customer's country (e.g., USA)                                 |
| **signup_date**         | Date customer joined platform (e.g., 2020-05-12)               |
| **last_purchase_date**  | Most recent purchase date (e.g., 2023-08-10)                   |
| **cancellations_count** | Number of cancelled orders (e.g., 2)                           |
| **subscription_status** | Active, Cancelled, Paused                                      |
| **order_date**          | Order placement date (e.g., 2023-06-15)                        |
| **unit_price**          | Price per unit (e.g., 49.99)                                   |
| **quantity**            | Number of units purchased (e.g., 3)                            |
| **purchase_frequency**  | Orders per year (e.g., 15)                                     |
| **preferred_category**  | Most purchased category (e.g., Electronics)                    |
| **product_name**        | Product name (e.g., Smartphone)                                |
| **category**            | Product category (Electronics, Clothing, Home, Beauty, Sports) |
| **gender**              | Customer gender (Male, Female, Other)                          |
